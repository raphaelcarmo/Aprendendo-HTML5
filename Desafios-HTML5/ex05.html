Tags usadas neste exercício:

  1. Tag para deixar texto em negrito <strong>

  2. Tag para links de outros sites <a href="link do site" target="_blank" rel="external">
    1. href serve para colocar o link do site que o usuário vai clicar.
    2. target="_blank" vai abrir outra aba para o site que você clicou no link em href.
    3. rel="external" Indica que o link não faz parte do site, ou seja é um link externo.

    Quando usa a tag <a> para links externos, é adequado usar esses 3 atributos acima.

  3. Tag para adicionar imagem <img src="local onde esta a imagem" alt="descrição da imagem">
